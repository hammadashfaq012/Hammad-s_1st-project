# 🏡 Architecture Website

A modern and responsive architecture website built using HTML, CSS, Bootstrap, and JavaScript.  
This project allows users to explore custom house designs, view property layouts, and experience a clean UI with authentication pages.

---

## ✨ Features

- Modern Responsive Design
- Property Showcase
- Custom Home Designs
- Login & Signup Form
- Smooth User Interface
- Interactive JavaScript Components
- Mobile Friendly Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap
- JavaScript
